CSS Rule

verticle-align:

ใช้กำหนดตำแหน่งของ object ในแนวตั้ง

z-index:

จัดให้ content ที่ซ้อนกันเรียงลำดับหน้า-หลัง

border-image:

ตั้งให้ชนิดของ border เป็นรูปที่เรากำหนดไว้

border-image-slice:

ใช้กำหนดความถี่ของกรอบ - ในเว็บนี้ใช้เพื่อยืดรูปของกรอบให้ชิดกับมุมทั้ง 4 ด้าน

list-style-position:

ใช้กำหนดตัว list ว่าจะอยู่ข้างใน หรือข้างนอก table

border-spacing:

เว้นระยะห่างของ border ข้างนอกและข้างใน ใช้กับ table

border-collapse:

กำหนดให้กรอบของ table ที่อยู่ติดกันนำมารวมกัน
